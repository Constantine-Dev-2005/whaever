1. АРХІТЕКТУРА КАСКАДУ (8 СЕКЦІЙ К1804ВС1)
───────────────────────────────────────────
Розрядність       : 32 біти (8 секцій × 4 біти)
Каскадування пер. : Послідовне. Cn+4(сек. i) → Cn(сек. i+1)
Тактування        : Спільний CP (CLK, пін 19) для всіх 8 секцій
Оцінка затримки   : T_min = 8×t_carry + t_setup + t_skew
                    = 8×20 нс + 25 нс + 5 нс = 190 нс → f_max ≈ 5 МГц (консервативно)

4. РОЗРАХУНОК НОМІНАЛІВ ПАСИВНИХ КОМПОНЕНТІВ
─────────────────────────────────────────────
4.1 Блокуючі конденсатори C1–C8
  Мета: придушення ВЧ-перешкод на лінії VCC під час фронтів CLK.
  Формула антирезонансу: f_антирез = 1 / (2π × sqrt(L_паразит × C))
  Параметри: L_паразит ≈ 10 нГн (монтажна індуктивність при DIP-40)
             C = 100 нФ (X7R — стабільна ємність до 125°C)
  Результат: f_антирез = 1 / (2π × sqrt(10e-9 × 100e-9))
           = 1 / (2π × 31.6 нс) ≈ 5 МГц ≥ f_max = 5 МГц — прийнятно.
  Примітка: для збільшення запасу рекомендується два 100 нФ паралельно (L_eff = L/2),
  що підвищує f_антирез до ≈ 7 МГц > f_max.
  Вибір ESR: ESR < 0.1 Ом (тип X7R 0805 — ESR ≈ 0.05 Ом).
  Розміщення: між виводами 1 (VCC) і 20 (GND), відстань від МС ≤ 3 мм.

4.2 Груповий конденсатор C9
  Мета: фільтрація низькочастотних пульсацій від джерела живлення.
  Номінал: 10 мкФ / 16В (коефіцієнт запасу: 16В / 5В = 3.2).
  Тип: алюмінієвий електролітичний (або танталовий для кращих ВЧ-характеристик).
  Розміщення: безпосередньо на вводі живлення плати (XP1, пін 1).

4.3 Перемички /OE (піни 18 всіх DD1–DD8)
  Режим: /OE = LOW (GND) → виходи F[3:0] та Y[3:0] завжди активні.
  Реалізація: перемичка 0 Ом (або пряме з'єднання) між піном 18 та шиною GND.
  Альтернатива (для керованого 3-стану): підключити /OE до BUS_EN через
    інвертор К1533ЛН1 (I_ol ≤ 8 мА при V_OL ≤ 0.4 В — задовольняє потребу).

4.4 RC-фільтр на CLK (рекомендовано)
  Мета: придушення EMI на тактовій лінії перед входами CP(19).
  Схема: послідовний R = 33 Ом + паралельний C = 47 пФ (до GND) на кожному вводі.
  При f_max = 5 МГц: T_CLK = 200 нс; T_CLK/10 = 20 нс.
  Час наростання: tau = R × C = 33 × 47e-12 ≈ 1.55 нс << 20 нс (T_CLK/10) → допустимо.

5. КЕРУВАННЯ АЛП — ПОЛЕ МІКРОКОМАНДИ I[8:0]
─────────────────────────────────────────────
I[2:0] (піни 22–24)  Вибір АЛП-операції:
  000 ADD    001 SUBR   010 SUBS   011 OR
  100 AND    101 NOTRS  110 EXOR   111 EXNOR

I[5:3] (піни 25–27)  Джерело операндів A/B:
  000  A=R[A],   B=R[B]    — регістровий файл → АЛП
  001  A=R[A],   B=D[3:0]  — рег. A + прямий вхід
  010  A=0,      B=R[A]    — нульовий + рег. A
  011  A=0,      B=R[B]    — нульовий + рег. B
  100  A=0,      B=D[3:0]  — нульовий + прямий вхід
  101  A=D[3:0], B=R[B]    — прям. вхід + рег. B
  110  A=D[3:0], B=D[3:0]  — прямий вхід подвійно
  111  A=R[B],   B=R[B]    — рег. B подвійно

I[8:6] (піни 28–30)  Запис результату та зсув:
  000  F → R[B],    Y = F        — пряме записування
  001  F → R[B],    Y = A        — збереження A
  010  F → нікуди              — тільки обчислення
  011  F/2 → R[B] (зсув вправо) — арифм. зсув вправо
  100  2F  → R[B] (зсув вліво)  — арифм. зсув вліво
  101  SWAP Q, R[B]              — обмін Q/регістр
  110  F → Q                     — запис у Q-регістр
  111  F → R[A] і R[B]           — широке записування

6. ЗСУВНИЙ ЛАНЦЮГ RAM0/RAM3 ТА Q0/Q3
───────────────────────────────────────────
Зсув вліво  (SHL): RAM3(сек. i) → RAM0(сек. i+1), Q3(сек. i) → Q0(сек. i+1)
Зсув вправо (SHR): RAM0(сек. i+1) → RAM3(сек. i), Q0(сек. i+1) → Q3(сек. i)
Зовнішні вузли:
  Q0_IN       (DD1.пін 12)  — вхід зсуву вліво (або фіксований рівень)
  Q3_OUT      (DD8.пін 13)  — вихід зсуву вправо
  RAM0_IN     (DD1.пін 10)  — вхід знакового біту при арифм. зсуві
  RAM3_OUT    (DD8.пін 11)  — вихід до алгоритму ділення

7. ПІДКЛЮЧЕННЯ ДО БМУ (К1804ВУ4)

8. ОБОВ'ЯЗКОВІ ЗВ'ЯЗКИ НА СХЕМІ (ЄСКД ВІДПОВІДНІСТЬ)
───────────────────────────────────────────────────────
Згідно з ДСТУ ГОСТ 2.702 на схемі E3 ОБОВ'ЯЗКОВО відображати:
  ✓ Шина A[31:0], B[31:0] — двонапрямні, позначка /4 на відгалуженнях
  ✓ Шина D[31:0]  — прямий вхід, позначка /4 на відгалуженнях
  ✓ Шина I[8:0]   — вхід мікрокоманди, позначка /9
  ✓ Ланцюг переносу C0 → C4 → … → C32 (Cn та Cn+4 кожної секції)
  ✓ Зсувна шина RAM0/RAM3 — між секціями та зовнішні входи/виходи
  ✓ Зсувна шина Q0/Q3    — між секціями та зовнішні входи/виходи
  ✓ CLK (CP, пін 19)      — до всіх 8 МС
  ✓ +5В та GND            — шини живлення з позначками
  ✓ /OE → GND             — позначка або символ на схемі
  ✓ C1–C8 (або посилання на перелік елементів)
  ✓ XP1 — роз'єм живлення
  ✓ Позиційні позначення DD1–DD8, C1–C9, R1–R8, XP1
  ✓ Шини F[31:0] та Y[31:0] — результатні виходи
  ✓ OVR (DD8.пін 39) та Cn+4 (DD8.пін 40) — статусні виходи до БМУ

9. СУМІСНІСТЬ З ЕЛЕМЕНТНОЮ БАЗОЮ (К1533, К537)
─────────────────────────────────────────────────
К1804ВС1 (TTL Schottky):  V_OH ≥ 2.4 В, V_OL ≤ 0.5 В, I_OL = 20 мА
К1533хх  (LSTTL):         V_IH ≥ 2.0 В, V_IL ≤ 0.8 В → пряма сумісність
К537РУ10 (SRAM):          V_IH ≥ 2.0 В → пряма сумісність
Підтяжка (якщо потрібна): R = (VCC − V_OH_min) / I_load
  При V_OH = 2.4 В, I = 0.2 мА: R = (5.0 − 2.4) / 0.0002 = 13 кОм → обираємо 10 кОм
