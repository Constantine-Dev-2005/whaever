ТЕХНІЧНА СПЕЦИФІКАЦІЯ: БЛОК МІКРОПРОГРАМНОГО УПРАВЛІННЯ (БМУ)
(На основі курсового проекту КВ-32)

1. АРХІТЕКТУРА БМУ
- Контролер: К1804ВУ4 (Формувач адреси мікрокоманд).
- Мікропрограмна пам'ять: КР556РТ6 (512 слів).
- Керування: Генерація послідовності мікрокоманд на основі КОП та стану АЛП.

2. ЗВ'ЯЗКИ БМУ - БОД
- Адресація: ФАМ керує 9-розрядною адресою мікропрограмної пам'яті.
- Аналіз умов: Сигнал знаку АЛП (біти 31 РЗП) подається на вхід CC (Condition Code) контролера К1804ВУ4.
- Лічильник циклів: Використовується вбудований лічильник ФАМ для ітерацій ділення (ініціалізація 32).

3. МІКРОПРОГРАМА ДІЛЕННЯ (ТАБЛИЦЯ 7.1 - ЛОГІКА БМУ)
- 00H -> 01H: Початок, ініціалізація.
- 01H -> 02H: Розрахунок знаку (XOR умов).
- 02H -> 03H: Старт циклу зсуву.
- 03H/04H -> 05H: Виконання арифметики з переходом за умовою (CC).
- 05H: Перевірка лічильника (Counter != 0 -> 02H, інакше -> 06H).
- 06H: Фіналізація.

4. СИНХРОНІЗАЦІЯ
- Тактування: Синхронно з БОД через спільний генератор CLK.
